# Proposal: Stock Analysis Report Generator

## Intent（意圖）
建立一個可產生個股分析報告的模組，將已匯入的價格資料與後續擴充的基本面、新聞、籌碼資料整合成一份可讀、可驗證的分析摘要，作為 AI 投資分析師的核心輸出之一。

## Scope（範圍）
### In scope
- 定義第一版個股分析報告輸出結構
- 建立 Python 報告生成器骨架
- 先以 canonical 價格資料為主，產出價格趨勢摘要
- 支援指定股票代號（第一版先以 2330、2454 為驗證目標）
- 預留月營收、財報、新聞、法人資料的擴充欄位

### Out of scope
- 完整投資建議或自動下單
- 前端 UI 呈現
- 複雜自然語言事實查核引擎
- 完整技術指標系統（後續可再加）

## Approach（方案）
第一版採兩層結構：
1. `stock_analysis_context`：先從資料庫彙整個股分析所需的結構化資料。
2. `stock_report_generator`：根據結構化資料產出報告文字。

第一版先聚焦價格趨勢，包含最新收盤、近 N 日變化、資料來源與簡單趨勢描述；後續再逐步接上財報、月營收、新聞與法人資料。
